# [화면설계서] 임플란트 상품 상세 화면 (Implant Product Detail)

## 1. 개요
- **화면 ID**: SCR-PROD-DET-001
- **화면명**: 임플란트 상품 상세 화면
- **설명**: 오스템 임플란트(TS II SA 등)의 상세 정보를 확인하고, 규격별 옵션을 선택하여 주문서에 담는 화면

## 2. 화면 구조 (Layout)
### 2.1. 상단 레이아웃 (Product Head)
- **좌측 (ProductGallery)**: 상품 이미지 영역 (450x450 메인, 82x82 썸네일 리스트)
- **우측 (ProductInfo)**: 상품 기본 정보 및 옵션 선택 영역

### 2.2. 하단 레이아웃 (Product Bottom)
- **DetailTabs**: 상품상세, 정보확인, 리뷰(건수), Q&A(건수) 탭 구성

## 3. 상세 기능 정의

### 3.1. 상품 기본 정보 (Head Info)
- **브랜드/상품명**: 상품의 전체 명칭 표시 (예: TS II SA Implant NoMount)
- **별점 및 리뷰**: 평점과 리뷰 건수 표시 (클릭 시 리뷰 탭으로 이동)
- **가격 표시**: 
  - 일반 판매가 표시
  - '임플란트 PKG' 버튼 클릭 시 가격비교 팝업 노출

### 3.2. 계단식 규격 선택 (Cascading Option Selection)
임플란트 상품의 특수성을 반영한 3단계 선택 로직을 적용한다.
1. **1단계 (Size)**: Mini / Regular 중 선택
2. **2단계 (Diameter)**: 
   - Size 선택 전에는 비활성화
   - Mini 선택 시: 3.5mm만 활성화
   - Regular 선택 시: 4.0, 4.5, 5.0mm 활성화
3. **3단계 (Length)**: 
   - Diameter 선택 전에는 비활성화
   - 3.5mm 선택 시: 6.0, 7.0mm는 불가능(X 표시) 처리
   - 최종 Length 선택 시 하단 '선택상품 리스트'에 즉시 추가

### 3.3. 선택 상품 리스트 (Selected Product Table)
- **품목코드/규격**: 선택된 옵션의 SKU 정보 표시
- **수량 조절**: +, - 버튼을 통한 수량 변경 (최소 1개)
- **개별 금액**: (단가 x 수량) 실시간 계산 표시
- **삭제**: 리스트에서 해당 품목 제거

### 3.4. 보험코드 조회 (Insurance Code Lookup)
- **조회하기 버튼**: 클릭 시 팝업(Layer Popup) 노출
- **팝업 내용**: 해당 상품군에 속한 모든 규격별 보험코드 매핑 테이블 제공

### 3.5. 예상 결제 금액 및 버튼
- **총 수량/총 금액**: 선택된 모든 상품의 합계 표시
- **관심상품**: 스크랩 기능
- **주문서 담기**: 선택된 상품들을 장바구니로 전송 (상품 미선택 시 경고 노출)

## 4. 데이터 인터페이스
- **API (GET)**: `/api/products/{id}` - 상품 기본 정보 및 메타데이터
- **API (POST)**: `/api/cart` - 선택 상품 리스트 전송

## 5. 예외 처리
- **옵션 미선택 시**: "규격을 선택해주세요" 안내 토스트 노출
- **품절 상품**: 옵션 버튼 비활성화 및 '품절' 문구 표시
