# [VOC 분석 보고서] 차세대 오스템몰 개선 기획안

본 보고서는 2026년 4월 현재 접수된 '차세대 오스템몰' 관련 영업 VOC 데이터를 기반으로, UX 디자인의 거장 도널드 노먼의 디자인 원칙과 닐슨 노먼 그룹(NN/g)의 사용성 휴리스틱을 적용하여 분석한 결과입니다.

---

## 📅 현황 요약 및 통계
*   **분석 대상**: 구글 스프레드시트 내 '접수 내용' 열 (약 140건)
*   **주요 이슈 범주**:
    *   **시스템 오류 (42%)**: 검색 정확도 저하, 주소 검색 실패, 결제 단계 프리징.
    *   **UX/UI 편의성 (35%)**: 뒤로가기 시 목록 초기화, 모바일 레이아웃 깨짐, 상품 식별 어려움.
    *   **콘텐츠/정보 (15%)**: 거래명세서 출력 지연, 상품 이미지 상이, 품목코드 누락.
    *   **로그인/권한 (8%)**: 계정 통합 이슈, 영업사원(BO) 접근 권한 복잡성.

## 🧠 도널드 노먼의 디자인 6원칙 분석
| 원칙 | 현장 목소리 (VOC) | 기획적 해결안 (To-Be) |
| :--- | :--- | :--- |
| **가시성 (Visibility)** | "이게 DDS 상품인지 외부 상품인지 한눈에 안 보여요." | 상품 카드 내 명확한 유형 배지(Badge) 및 필터 강화 |
| **행동 유도성 (Affordance)** | "어디를 눌러야 정산 내역 페이지로 가는지 헷갈려요." | 주요 기능 버튼의 시각적 강조 및 계층 구조 재정의 |
| **피드백 (Feedback)** | "주문 버튼 누르고 나서 로딩만 돌고 결과가 안 나와요." | 실시간 진행 상태 스켈레톤 UI 및 명확한 성공/실패 토스트 알림 도입 |
| **제약 (Constraints)** | "가격을 다른 사람한테 안 보여주고 싶은데 조절이 안 돼요." | 개인 설정 내 '가격 노출/숨김' 모드 스위치 제공 |
| **대응 (Mapping)** | "정렬 순서가 가격순인데 뒤죽박죽인 것 같아요." | 정렬 엔진 로직 고도화 및 정렬 기준에 대한 시각적 매핑 강화 |
| **일관성 (Consistency)** | "브라우저마다 메뉴 위치가 다르게 보여요 (IE11 이슈)." | 반응형 그리드 시스템 정규화 및 크로스 브라우징 완벽 지원 |

## 🛠 닐슨 노먼 그룹(NN/g) 10대 휴리스틱 평가
1.  **시스템 상태의 가시성**: 장바구니 개수와 실제 담긴 상품 수 불일치 해결 필요.
2.  **시스템과 실제 세상의 일치**: '정산' 대신 사용자가 더 익숙한 '매출/결제 확인' 등의 용어 혼용 검토.
3.  **사용자 제어 및 자유**: **(최우선)** 상세페이지 확인 후 '뒤로 가기' 시 필터링/스크롤 위치 유지 (State Preservation).
4.  **일관성 및 표준**: 모바일 웹에서 텍스트 영역을 벗어나는 이미지 크기 고정.
5.  **에러 방지**: 주소 입력 시 구 주소/신 주소 검색 엔진 통합 및 유효성 검사 강화.
6.  **회상보다는 인식**: 최근 검색한 품목코드 자동완성 및 추천 상품 노출.
7.  **유연성 및 효율성**: 영업사원을 위한 '다량의 품목코드 일괄 업로드 주문' 기능 제안.
8.  **미학적이고 미니멀한 디자인**: 복잡한 주문 내역 페이지의 정보 위계(Info Hierarchy) 재설계.
9.  **에러 인식 및 회복**: "서버 오류" 대신 "검색 결과가 없으니 품목코드를 다시 확인하세요"와 같이 구체적 가이드 제공.
10. **도움말 및 문서化**: DDS 회원 가입 절차에 대한 대화형 가이드(Walkthrough) 도입.

## 🚀 기획자 관점의 핵심 개선 로드맵

### 1단계: 즉시 개선 (Short-term / Hot-fix)
*   **뒤로가기 브라우저 캐싱**: 검색 결과 리스트 위치 및 필터 상태 값 저장 로직 구현.
*   **품목코드 정규화**: 하이픈(-) 포함 여부와 상관없이 동일 품목이 검색되도록 검색 엔진 튜닝.

### 2단계: 중기 과제 (Mid-term / Feature Update)
*   **상품 식별 시스템**: 상품 썸네일에 오스템/DDS/외부몰 구분을 위한 워터마크 또는 라벨 일괄 적용.
*   **모바일 UX 고도화**: 장바구니/주문 페이지의 폰트 사이즈 및 버튼 클릭 영역(Tap Target) 최적화.

### 3단계: 장기 전략 (Long-term / Strategy)
*   **디지털 물류 정보 연동**: 배송 업체 API 실시간 연동을 통한 '배송 전-배송 중-완료' 단계의 시각적 타임라인 제공.
*   **영업사원 특화 대시보드**: 거래처별 선납금 잔액 및 최근 구매 빈도 분석 기능 추가.
